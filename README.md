# gff-viewer

Interactive GFF/GTF annotation viewer with feature type coloring, attribute parsing, and region statistics.

## Features

- Auto-detects GFF3 and GTF formats
- Feature type color coding (gene, mRNA, exon, CDS, etc.)
- Column header sorting (click to sort ascending/descending)
- Text search across all fields
- Chromosome and feature type dropdown filters
- Strand color coding (+ green, - red)
- Attributes column parsed as key=value pairs
- Computed region length column
- Summary statistics (feature count, chromosomes, feature type distribution)
- Pagination for large files (100 rows per page)

## Supported Extensions

- `.gff`
- `.gtf`
- `.gff3`

## Installation

Install from the **Plugins** tab in the AutoPipe desktop app.
